# tpfinal-concurrente

COMO SIMULAR EL PARQUE DE ATRACCIONES

SEGUIR LAS SIGUIENTES CONDICIONES EN ORDEN DE IMPLEMENTACION PARA UN PROGRAMA MAIN DONDE PODRA EJECUTAR EL PROGRAMA (DE TODAS MANERAS HAY
UN PROGRAMA MAIN DE EJEMPLO PARA EL USUARIO)

CONDICIONES:
1: Crear una instancia de Parque.java (recurso compartido para todos los visitantes) -- Dentro de este objeto se crearán automáticamente instancias de todas las atracciones
2: Crear Instancias de las clases correspondientes a los hilos que simulan las atracciones necesarias (simulacionMontaña, simulacionAutosC).
3: Crear el hilo Hora el cual se encargara de ir cambiando la hora.
4: Crear instancias de los asistentes del espectaculo.
5: Crear instancias de la clase persona y arrancar dichos hilos.
